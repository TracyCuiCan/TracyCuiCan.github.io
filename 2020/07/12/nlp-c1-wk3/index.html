Notes for NLP Course1 Week2 & 3

Naive Bayes & Word Embedding

Naive Bayes

  • Bayes rule
  • Log likelihood
  • Laplacian smoothing
  • For sentiment analysis, calculate prior as the the number of positive and negative documents,
    calculate conditional probablity as the frequency of the word in positive and negative documents.

Application of Naive Bayes

  • Sentiment analysis
  • Author identification
  • Information retrieval
  • Word disambiguation

Naive Bayes Assumptions

  • Independence: Not true in NLP
  • Relative frequency of classes affect the model, data is often imbalanced.

Word Embedding

Linear algebra operations in Numpy

Difference between lists and numpy arrays:

The + operator on NumPy arrays perform an element-wise addition, while the same operation on Python lists results in a list concatenation, same with the * operator

The product operator * when used on arrays or matrices indicates element-wise multiplications. Do not confuse it with the dot product.

Find norm of a nparray or matrix: np.linalg.norm(X)

axis=0 in params performs column wise operations while axis=1 in params performs row wise operation.

Vector space

Similarity between two vectors can be calculated by Euclidean distance or cosine similarity, cosine similarity is prefered.

  • Euclidean distance: np.linalg.norm(A, B)
  • Cosine similarity: np.dot(A, B) / np.linalg.norm(A) / np.linalg.norm(B)

PCA for dimension reduction.

np.cov(m, rowvar=True) to calculate the covariance matrix, rowvar is True by default, then each column is a sample and each row is a feature.

np.linalg.eigh(a) to calculate eigen value and eigen vector. If a is symmetric, using np.linalg.eigh() is much more performant than np.linalg.eig().

np.argsort() sorts the values in an array from smallest to largest, then returns the indices from this sort. To apply the sorted index row wise, do X[indices_sorted], to apply the sorted index column wise, do X[:, indices_sorted].

X[::-1] to reverse the order of a list.
