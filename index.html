nlp-c2-wk3

N-gram

A sequence of N words. N-gram probability \(P(\omega_N|\omega_1^{N - 1}) = \frac{C(\omega_1^{N - 1}\omega_N)}{C(\omega_1^{N - 1})}\), and notice that \(C(\omega_1^{N - 1}\omega_N) = C(\omega_1^N)\).

When preprocessing sentence into N-gram models, we need to add character that mark the start and end of the sentence with <s> and </s> separately. For example, for n-gram preprocessing, tokenized_sentence = ["<s>"] * (n - 1) + tokenized_sentence + ["</s>"].

Probability of a sentence

Apply the chain rule, that \(P(A, B, C, D) = P(A)P(A|B)(C|A, B)P(D|A, B, C)\), this exact formula has the limitation that the long subsequences of our testing sentence are almost never represented in the training corpus. To approximate sequence probability, we apply Markov assumption: only last N - 1 words matter, that is, \(P(\omega_n|\omega_1^{n - 1})\approx P(\omega_n|\omega_k^{n - 1})\) where k = n - N + 1

Generative Language Model

Once you have the n-gram probability model, you can use it to generate text by choosing sentence start -> choose next n-gram starting with the last n - 1 word -> continue until <\s> is picked.

Perplexity

Perplexity is a measure of complexity in a sample of texts like how complex that text is. Smaller perplexity means better model, good language models have PP from 20 to 60.
$$PP(W) = P(s_1, s_2, …, s_m)^{-\frac{1}{m}}$$

Log Perplexity
\(\log PP(W) = -\frac{1}{m}\sum_1^m\log_2(P(\omega_i|\omega_j))\) where j = i - 1.

Out of vocabulary word (OOV)

OOV means unknown word, a special tag <UNK> is used to represent OOV in corpus and in input. To create a vocabulary V, you can use different criterias:

  • Min word frequency f
  • Max size of V, include words by frequency

NOTE: When comparing perplexity, only compare LMs with the same V.

Handle missing n-grams

  • Smoothing
    – Add one smoothing
    – Add k smoothing
  • Backoff, if N-gram missing => use (N-1)-gram.
  • Interpolation
    $$\hat{P}(\omega{n-2}\omega{n-1}) = \lamda x P(\omegan|\omega{n-2}\omega_{n-1}) + \lamda_2 x P(\omegan|\omega{n - 1} + \lamda_3 x P(\omega_n)$$
    and \(\sum_i \lamda_i = 1\)

HMM

A hidden Markov model has States, Transition matrix (including initial probability) and Emission matrix (from a hidden state to an observable state).

Viterbi Algorithm

Viberbi algorithm can be used to find the mostly likely sequence of a hidden states. The implementation can be decomposed into three main steps:

  • Initialization - Initialize best_probabilities and best_paths matrices that will be populated in feed foward path.
  • Feed Forward - At each step, calculate the probility of each path happening and the best paths up to that point.
  • Feed Backward - Start from the end of the sequence, find the best path with the highest probabilities.

For a given word, find all the “words” that are one/two edit distance away from it; filter out those not in the vocabulary; order by the probability of each word by it’s prior probability of appearing in the context; choose the one with the highest probability.

Machine Translation and Document Search

Frobenius Norm of matrix A = np.squrt(np.sum(np.square(A)))

To translate English to French by using embedding, we need to approximate the transformation matrix R that transforms the English embedding matrix X to the French embedding matrix Y. We can use gradient descent to find matrix R where argmin_R ||XR - Y||_F. After finding R, during inference process, we need to find K nearest neighbors of Y_hat in French vocabulary. When there are so many candidate neighbors (the whole vocabulary), KNN is very expensive. And Locality sensitive hashing comes into play.

Locality sensitive hashing defines multiple (N) planes that divide the space into 2^N hash buckets. Planes are represented by a vector that marks the direction in which you find the positive side of the plane. Given a embedding, if the sign of it’s product with the plane vector is positive, it’s on one side of the plane, this way we can hash the embeddings into buckets. The hash value of an input vector hash_value = 2^i * h_i where h_i = 1 for positive sign h_i = 0 for negative sign for the ith plane.

np.where(condition, x, y) returns elements chosen from x or y depending on condition. e.g. A = np.where(B >= 0, 1, 0) where A B are both matrixs, and A[i, j] equals to 0 or 1 depending on the value of B[i, j].

Difference between lists and numpy arrays:

The + operator on NumPy arrays perform an element-wise addition, while the same operation on Python lists results in a list concatenation, same with the * operator

The product operator * when used on arrays or matrices indicates element-wise multiplications. Do not confuse it with the dot product.

末日杂念

纽约疫情加剧，剧情发展到跟一个月前的国内角色互换，我大部分时间被困在家里，生活像被按了下shift键。上下班的界限变的模糊，有的时候可能吃完晚饭还要工作一下。很奇怪在家办公竟然觉得更累，大概是因为一陷进工作里就忘了起身忘了喝水。代码不知道怎么写的时候就在屋子里走来走去，走到卧室翻倒在床上，三秒后劝自己”let’s get on with it!”。有点担心精神状态垮掉，还好有跟纽约的小伙伴交流防疫的最新信息，同事之间一起云吃饭，还有国内小伙伴每天的关心。坏处是开始嫌弃自己的厨艺，研究菜谱的热情只维持在开始的两天，现在还在好好做饭，很可能再过两周吃饭会等于填饱肚子而已。好处是天还没黑的时候就可以出门跑步，室外已经是春天的温度了，风是暖的很舒服；以及可以在家放肆地听歌，虽然经常因为打扰到工作而暂停…

因为已经从国内看了一个多月的预告片，我对于这些生活变化的接受很顺利，早就在mindset里面了，完全没感到困扰。跟着大众买口罩买酒精囤日用品囤粮食，竟然有了一些本不该有的麻木。对待病毒，从轻视到恐慌再到理性对待，担心过病毒大规模爆发，自己被感染，纽约的医疗设施无法接收这么多病人，我作为外国人一定会优先靠边站，没办法回国，爸妈又来不了美国，连最后一面都见不了，这大概是最惨的状况了吧。也因为各大送菜app都暂停配送服务而panic shopping，去平时很少去的楼下超市买了一堆硬核粮食，以至于被收银员嘲笑，回到家之后自己也觉得好笑没出息，但至少安心。

疫情的消息铺天盖地，新闻媒体想要给我最快最客观最全面的报道，这样很好，感谢所有致力于把信息带给每个人的工作者们，但有时候吧我也想喘口气。哦听到了意大利封国，哦知道了美国对欧洲travel ban，哦病毒可能会造成肺部永久性损伤，是这样吗？好的。大家都在讨论各国采取什么政策来应对什么样的国情，我由于我自己的愚蠢对此没有观察没有想法也并不太关心，更关心的却是生活里的小事，去办公室拯救我的两盆草，同事们分享的WFH-lunch集锦，WFH-pets集锦，意大利人民在阳台上拿出自己五花八门的乐器绝学合唱国歌，同样是末日囤货，有人买可乐汽水、有人如我买大米和面粉、有人直奔罐头食物，为什么都在抢厕纸？switch玩什么游戏好…如果人类被这个病毒搞到灭绝，我想的记得的全是这些琐事，也不知道生命安全都受到威胁了，还想着午饭吃什么工作怎么做好这些有什么用。我已经俗到底了。

生活什么时候能恢复正常呢？什么样的生活是正常的呢？我不是一直这样工作吃饭，周末在家宅到天荒地老吗？好像没什么不一样。但我也知道，看不到的地方有很多人在无声的战场上，医务工作者们、研究人员、政府职员、楼下超市的补货员、送菜app的外卖小哥，我家里有水有电有wifi有食物，这么简单的平常是很多人的关照，感谢他们！我们无法作为个体来对抗这个未知又凶险的病毒，我们依附于文明，依附于国家政体，依附于制度，再单纯也要依附于集体。

昨天听完 The Daily 的 Special Episode - A Bit of Relief 有点感动，大概因为跟我一样，发现在危机和恐慌之下，回归生活细节最能给人以安慰和镇定，最后一个片段里读了 “On Living in an Atomic Age” by C.S. Lewis，贴出来分享一下：
“Believe me, dear sir or madam, you and all whom you love were already sentenced to death before the atomic bomb was invented: and quite a high percentage of us were going to die in unpleasant ways.”
“If we are all going to be destroyed by an atomic bomb, let that bomb when it comes find us doing sensible and human things—praying, working, teaching, reading, listening to music, bathing the children, playing tennis, chatting to our friends over a pint and a game of darts—not huddled together like frightened sheep and thinking about bombs. They may break our bodies (a microbe can do that) but they need not dominate our minds.”

跑步去买奶茶

还是跑步轶事，有一天心血来潮我决定去LIC那家奶茶店去买脏脏茶喝，跟二老一路跑一路等，被问了好几次怎么还没到之后，终于买到了！回程边喝边走，真的超开心，只是以后每次我再做这件事都会有一点点点不开心吧…

两个“土匪”

收拾行李的时候，老崔一直试图拿走我的冰酒和清酒，老冯一直试图拿走我的香水和鞋，我打趣说“你俩就是两个土匪，吃完喝完还要带走！”

健身达人

今年楼下健身房翻修，24小时开放不用再找门卫开门了，所以两个人白天在家没事的时候找机会就回去楼下健身，器械都试个遍，比我还6，每天回到家给我炫耀嘲笑我今天又不去运动，真烦人！

阳台会塌

在没学会去健身房之前，老崔白天在岛上暴走，老冯晚上在阳台跟着视频跳健身操/舞，有一次我看到跟她说“妈你注意点，别把阳台跳塌了”，于是冯老师再也没有去阳台上跳过了…

人家是不是不理你了

老冯的亲戚给我介绍了一个相亲对象，认识了几天之后二老关心一下进展，我说我们交换了照片，说完老崔问“是不是发完照片，人家就不理你了”，execuse me??有这么嫌弃自己闺女的吗？？后来跟老崔下楼买菜的路上我就反复逼问，老崔在“嘿嘿”尬笑了好几下之后躲不过，吞吞吐吐跟我说“人家看你在国外，以为你是国外留学的样子” “国外留学什么样子？” “高端一点” “我不高端吗？” “你看你那照片里穿个t恤…人家应该喜欢穿正装的”，这个逻辑你懂吗？反正我跟老冯都没懂。

吃

因为吃跟老冯吵了好多架，在Vegas为了吃InNOut跟着Google map走半天结果走到高速公路边只好无功折返，临走之前想去吃自助，回到纽约带他们去Nyonya吃马来西亚菜，每次我都能跟冯老师吵起来，她每次都是“崔粲，你怎么这么好吃，都已经很胖了！”，我真的好委屈…

没有人洗袜子了

假期快要结束的时候，依依惜别的情绪已经开始发芽，每次洗完澡把袜子丢在浴室门口的时候，我都要朝正在看电视的老冯痛呼一声“妈！你走了以后就没人给我洗袜子了！”，是啊，真的很让人绝望啊…今天晚上就要自己洗袜子了…

近日气温已经在26/7度，早晚已经有些凉了，窗外的叶子开始泛黄，夏天又悄悄过去了。这个夏天我们仨一起去了一些地方做了很多事情留下了很多美好回忆，我坐在这里间歇抹着眼泪敲完这些字，不得不为今年夏天画上句号了。回到一个人的生活，期待下一个夏天…

一八一九

又到了写年终总结的日子，早上下了一点点雪，窗外天阴沉沉的。今年有颖姐来陪我过新年和圣诞，每天逛吃太过开心就想这么稀里糊涂地迎接新年也不错，新年跟旧年应该也是差不多…但又隐约担心如果没有合理总结过去的这一年，新年会过得不好。回头一看18年真的是很充实的一年，好像把17年应该发生的事情都往后挪了，而且今年做的事情似乎能被月份很好的分割，所以决定也按月份来写。

1月 & 2月 —— 黑暗中行走的人和她的星光与灯塔

在继17年底面试的接连失利之后，浮躁终于都被打落，开始踏实下来。每天下了班回到家稍微休息，8点准时坐到书桌前打开leetcode，因为客厅暖气太冷，把桌子搬到了卧室墙角，一转身就是床，窝在那一小寸地方，不问不想，只在乎今天做了几道题。那段时间其实非常迷茫，17年收到的几个面试也都面完并挂掉了，下一步不知道要去哪里，转发各种转运帖子，最烦听到“再接再厉”。花了很长时间准备的ML面试看起来不会用得上，内心一直逃避的刷题跳槽这条路似乎已经不得不走。怪自己之前太天真，对于结果也不敢再去想，经常睡不着半夜起来嗑dreamwater，惶惶度日不敢问前程。但好在这黑暗的日子里有朋友的鼓励，去三藩面试的时候，蓝提出去过去陪我，我面试完我们在三藩逛博物馆，开车走一号公路，去唐人街吃小面，在沙滩上跳起来拍照，很感激。还因为《偶像练习生》这个节目知道了坤坤，人生第一个认真粉的idol，一下子明白了偶像存在的意义，理解了粉丝的全部心情，从上一个冬天到这一个冬天，坤坤一直是我的one pick!

3月 & 4月 —— 一点点收获与选择

3月跟4月开始赶鸭子上架地去面试。记得当时面完G出来，自我感觉特别差，五道题一道没做出来，两道跟面试官交流不好…回到家万念俱灰，因为这是最后一场面试了，手里仅有的一个选择是下下签，这意味着我几个月的面试之战全线溃败。但是在我伤心了两天之后，竟然过了，大概是之前的某个转运符起了点作用。可怀疑自己是擦线而过，一路谈offer都没有底气，拖着A巴结着G，提心吊胆一直到把名字签在那张offer letter上。也还是纠结了一下两个选择，A虽然企业文化和机遇不如G，但可以做ML，至少算是入了门。先转弯再加速还是先加速再找岔路转弯？本着A面试简单想去可以再面的原因，选了G。结果虽不尽理想但也还算不错，那天发了一条状态说“感觉像是从一个长长的隧道里走出来了，不管前方路况如何，just keep going!”，四五个月苦行僧一般的日子结束了!

5月 & 6月 —— 休息再休息一下

真正签offer是在5/11，然后就是H1B transfer，5月底去看了早就买好票的。买票的时候一切未卜，我当时想着不管怎样到五月底我应该所有的面试都结束了，有好结果就当庆祝，没有好结果就当中场休息，如今看来命运待我不错。然后就跟蓝一起去了趟夏威夷！蓝比我早到先去取车，我飞机刚落地，她就打电话问我喝什么饮料，拿行李出来见到她坐到副驾驶上，座位旁边一杯荔枝乌龙茶结了一层细密的水珠，天蓝蓝的，路前方仿佛能看到热浪，那种假期开启的轻松愉悦现在回想起来都觉得放松。十天的旅行，我们在酒店阳台看着大海喝冰饮料，拖着椅子和毛巾去沙滩上晒太阳，去molokini浮潜，在haleakala山顶看日落，因为一个累断腿的trail误了飞机，衣服里面永远穿着泳衣，后备箱永远放着人字拖，每天吃很多芒果和木瓜…夏威夷真的是天堂啊，有机会还要再去！

7月 & 8月 —— Family Time!

看完独立日的烟花我就前往机场迎接崔老师和冯老师了，拿着一盒洗好的尚且冰冰的车厘子。然后开启了两个月像傻子一样幸福的生活，二老陪着我经历离职入职的很多心路历程，关于这两个月的那些幸福的瞬间我都写在这一篇了。

9月 & 10月 —— 告别

据说真正的失恋不是分手那天，而是你得知前任有了现任的那天，我表示深切同意。这两个月就是我的失恋六十一天，其间经历的各种心情如今不想再去回望，就是一场痛苦的告别。跟旧爱告别，跟我脑海中的爱人告别，跟自己的执念告别，跟纯粹的初恋告别，跟对爱情天真的期望告别，跟一种可能的幸福生活告别…我仍然觉得不甘心但是却不后悔。我们之间有过一段很好的感情，都很用心地爱过对方，可又因为我们不够坚定最终没能走到一起，还是心里最亲近的人但已经感到陌生。世事未能如愿，就把回忆珍藏起来吧，祝我们都能过上想要的生活。

11月 & 12月 —— 重新出发

从隧道走出来就是坦途了吗？当然不是。从一个在组里带新人的senior回到啥也不懂兢兢业业被人带的junior，可是我早就没有刚毕业的junior心境了。这种不被看重，丝毫没有存在感的心情才是残酷的现实。适应新的公司文化和管理方式，做完全没做过的项目，跟新同事合作，再从头一点点build up自己的credibility。我很着急，但我也知道这需要时间而且过程可能并不如意。这是offer letter上隐形的附加筹码，需要在“跳槽成功”的喜悦退散过后才会慢慢看清的摆在面前的庞然大物，而我必须要越过它才能向前走。身边多了很多优秀的人，对比之下看到之前被忽视的不足，一切还在摸索中，希望在这个过程中能改掉一些毛病，学到一些东西，希望这个过程还顺利。

2018真的是很充实的一年，学到了一点坚持，又一次学到了一点放下，学会了在一个人死扛不了的时候向身边的人示弱求助。是一个人住的第二年，跟自己相处的很舒服。我觉得我找到了更多的自我，这让我变的更加坚定和自由，让我活地更放松了。感谢一路碰到的贵人们，感谢家人和朋友们默默的陪伴，特别鸣谢崔老师和冯老师。

时钟翻到2019的那一刻魔法不会发生，生活还是继续，没做的琐事还是要去做，没读完的书还要继续读。然而我相信接下来的365天会给我很多机会成为更好的自己，而我也要尽最大努力不去辜负它。内心的目标依旧，对自己对生活依然不设限。就继续前行吧，试着用更温柔的方式，留心更美的风景！

备份

两周前为了今天买的玫瑰，终于还是被我给养成了这个样子，百合菊花们都活的好好的，就数她矫情。当时为了这束花，你还吃醋，我为了气你不能跟我一个过情人节骗你说是别人送的，结果你还真较上劲了，这束一开始就没被看好的玫瑰果然从那天起每况日下，换盐水喷清水都不管用，义无反顾的枯萎了。
然后我们也分手了，我数不清是第几次，但我想狠下心让它成为最后一次，你再回来求我我也不会心软了。

早上说狠话的时候我心里想，这六年我无畏无悔，对你我也问心无愧。可后来仔细一想，我也并没有那么光明磊落，是我告诉你读完书就回去，是我又告诉你工作两年就回去，哪怕到了现在我说不知道几年才回去，你还是说等我。但我不想让你等我啊，为什么你不能来，那么多人都做得到，为什么你这么怂逼。这段感情从来都是我说了算的，只不过这一次你觉得自己办不到了，现在我觉得我有愧，但哪怕我这么觉悟了，也还是改不了我的决定。

你也不是我心里的最佳伴侣，只不过这些年在一起习惯了，所以才看后来的那么多人都不如你。你真怂，你想做自己把握内的事情，你的人生观跟我这么不一样，我们注定不能一起冒险。我一定今后遇到的人都比你强，你却再也找不到比我好的。找不到嫁不出去就算了，人生这么艰难，找到一个相看不厌共度一生的人那么难，我找不到也正常。恩，我现在还是这么想的，let’s see。

其实我想要的不多，两个人租一个一室一厅，养几盆花草，甚至过段日子养只猫猫狗狗，平时你上学我上班（我养你咯，暂时，但租的房子要离我公司近这是肯定的），周末天气如果好就开车出去玩（你带着我，我不要学开车），天气不好就窝在客厅沙发上看两部电影，看完交换一下观后感，心血来潮一起做顿好吃的，开瓶红酒，夏天夜晚坐在阳台慢慢喝。我知道你也想要，只不过我想让这些发生在纽约而你想让发生在成都，就这点差别。

大概是智商不太高的缘故，你的脑回路总是跟别人不太一样。你像只鸵鸟，高兴的时候脖子仰起四处张望，好神奇。遇到搞不定的事情，就把头埋在沙子里，假装没有这件事，一切照旧但就是对这件事只字不提。我前几天逼你做个选择，你两三天不理我，好不容易蹦出一句话，为我为什么喜欢你，说你自己就像你妹妹那个小男朋友似的，配不上我。我很生气，我气你这是哪门子逻辑，我们走到这一步一点都不是感情的问题，是现实，是我的不妥协和你的做不到。我还气你，两三天没消息，我威胁要拉黑你都震不出来，好不容易有了点反应，却是这个破问题！我以为你去拯救世界了好吗。

总之你还是那副死样子，我说那我们就再也不要联系，我受够了你的反复和纠结。刚说完你就把我也拉黑，大概是报复我先拉黑你，还有就是因为你又要逃避，我知道我说的那些话你一眼也不想多看。我真的受够了！可我知道你这么做就表示你并没有释怀，并没有想清楚这整件事情。但我不想解释了，你一旦钻进牛角尖，就刀枪不入五毒不侵，我奈何不了你，就随你去吧，反正分手从来就不是一件可以做的潇洒的事情。

我遗憾啊，自己养了六年的一头猪如今要被别人拿去宰了吃。可我想来想去也没有别的办法，我有我想要的生活，在你的计划里我会过的不开心。但我习惯了有你，哪怕这三年隔着半个地球的异国。我不是一个坚强的人，我也怂，可我这个怂到家的人偏偏能从你这个更怂到家的人那里获得生活的勇气，也是奇怪。我也无奈啊，该说的都说了该做的都做了，来不来还是你的决定，我一点办法都没有，你要是敢往前走一步我就敢走两步，但你要放弃。

这几年，别人约我去哪里玩或者吃好吃的，我都是拒绝，我不想跟他们一起，我想着等你来了我要跟你一起去，越长大越觉得很多事情要跟特定的人一起完成，不然一点意思都么的，还不如宅在家里，这也是我这几年越来越宅的原因。我想着要一起去很多地方，去吃好吃的东西，现在不要再等等，于是这么错过一个又一个，如今我们都分手了，却一件也没实现。从今往后，我要勇敢一点，没有人可等了，要学着自己去享受。

就这么多吧，写这篇也是因为洗澡的时候想起来，突然有话要说。又哪能说的完呢，留给时间吧。祝我们都过上理想的生活。
