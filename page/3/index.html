5.1 试述将线型函数\(y = \omega^Tx\)用作神经元激活函数的缺陷.
线型激活函数就等同于线性回归， 当样本不能线型可分的时候，不能很好的对样本分类。

5.2 试述使用图5.2(b)激活函数的神经元与对率回归的联系。
图5.2(b)是sigmoid函数，当使用sigmoid函数作为激活函数时\(y = \frac{1}{1 + e^{-(\omega^Tx + b)}}\)；用Logistic regression做分类的时候\(y = \frac{1}{1 + e^{-(\omega^Tx + b)}}\)；所以使用sigmoid函数作为激活函数的神经元，是在解决一个对率回归问题。

5.3 对于图5.7中的\(\upsilon\)试推导出BP算法中的更新公式5.13.
$$\Delta \upsilon_{ih} = - \eta \frac{\partial E_k}{\partial \upsilon_i}$$
$$ = \frac{\partial E_k}{\partial b_h} \frac{\partial b_h}{\partial \alpha_h} \frac{\partial \alpha_h}{\partial \upsilon_i}$$
其中$$e_h = - \frac{\partial E_k}{\partial b_h} \frac{\partial b_h}{\partial \alpha_h}$$
$$\frac{\partial \alpha_h}{\partial \upsilon_i} = x_i$$
所以\(\Delta \upsilon_i = \eta e_h x_i\).

读书｜理想主义者的出路

被这本书吸引是因为标题，最近在学着看画，书名让我很好奇，怎么在美术馆谈恋爱？我以为是一个长篇，一看发现还挺有来头，”最年轻的老舍文学奖得主最新文集”，友邻有三个人读过这本书还都给了五星，我更期待了。

想到这个题目是在地铁上，昨天花三个小时往返法拉盛为了吃一口馋了两周的牛肉面和奶茶，小书包里当然要背上kindle，回来的地铁上我在读那篇《西瓜》，读得我很难过，然后这个命题一下子出现在脑海里，几个故事就串起来了。

理想主义者？

在我看来文珍这本书写的是很相似的一群人，故事的主人公们普遍到了30岁这个坎，而立之年，在《普通青年宋笑在大雨天决定去死》这篇，宋笑有过嘲讽，而立意味着”该站起来了，可是，难道之前他一直都是趴着或者躺着的吗？那趴惯了又该怎么办呢？“。如果三十岁之前你还有些自由，觉得自己还是个没长大的孩子，倔强地不向生活妥协，想让自己心里的赤子肆意地撒欢；而生活和群众对你也还有宽容，甚至理想主义还能给你增添一点个人魅力。但到了三十岁世人对你的期待是该收起来了，回到现实，每天要考虑房子车子孩子了，不切实际的东西都不要想了。你已经到了人生的这个阶段，一切不合群不常规就会变得面目可憎，是没有意义并且可笑又可悲的了，甚至有人在等着看你笑话，你看你折腾这么久还是没我们“幸福”，自作自受。

三十岁像是来到悬崖边，跳下去是水深火热还是锦上繁花不知道，但是所有人都在催着你快跳啊，快做决定，看着前面的人扑通扑通或麻木或坚决地一个个跳下去了。这群人心里在想”为什么这个悬崖一定要跳呢？“”如果一定要跳，为什么我要被逼着现在跳呢？“。于是就有了这些困境，其实说他们是理想主义者也不算恰当，什么时候想要自由地选择人生不随大流成了一种理想主义？我后来想或许他们应该被叫做清醒者，因为还能意识到自己的处境并保持怀疑，但好像也不对，也有很清醒地决心积极入世并立志出人头地地跳下去的啊。但不管怎么称呼，你懂我意思，他们是这么一个弱势群体。

可是走出困境的出路呢？无外乎两种，一为决裂，一为和解。文珍也写了两种，这两种想法在书里来回振荡，每个读者都有他自己的答案。

决裂

你看《银河》里的苏令，”那一瞬我就把彼此暗淡无光的前路看了个清楚透亮，得一辈子往前跑，跑下去。停下来，庸碌的生活就会追上来，就会把我们拖入流沙底部。停下来就是个死。“ 他们两个私奔者从北京一路跑到塔县，还是没逃过生活无处不在的爪牙，最后只有一死。也不是回不去，不就是被人看看笑话，不就是承认自己输了，但回去大概还不如死了，活着从来也不比死了轻松。爱情多么美好，可当你想要的爱情无法被现实驯服，你选择献身还是放弃？

你看《录音笔记》里的曾小月，你以为自己小心翼翼，顺从，不打扰别人，就可以安静的随心生活了？到处都是眼睛到处都是声音。没有知己没人说话，她做了妥协的，那我就跟自己说话吧，想开了放开了也乐得开心。“达则兼济天下，穷就独善其身。”可以独善其身吗？人人都抱着拯救你的心态试图让你变得跟他们一样。如果是你你怎么办？

你看《我们究竟对不起谁》里的顾安，就是想要个幸福婚姻的普通女生而已，偏偏成了同妻，是那对相爱的gay错了吗？他们也很无奈，甚至还给婚外情创造契机，偏偏遇到一个穆斯林，那是穆斯林错了吗？他都在古兰经里找到支持自己行为的真理，打算打着发展信女的旗号去寻找真爱了，偏偏女主是个基督徒。怂就怂了，还要撒个谎说自己是gay，所以爱上同性这件事情会比爱上异教徒更冠冕堂皇更值得原谅吗？谁错了，谁对不起谁？“婚姻到底是什么？一双网购的多半不合脚的鞋？”，错的是那些认为你光着脚就不能走路，要穿一定要穿一双特定样子的鞋，不穿还要替你疼的人吧。

和解

可要是不想搞得那么难看呢，好死不如赖活着，那就试着和解吧。

《衣柜里来的人》小枚不就是，在跳下悬崖之前她还有些不甘心，于是回去再绕个弯，把另一种可能性试一下，死了心才能踏实。然而远方的意义大概就在于它是个美好的愿想，因为等你到了远方发现，跟家门口也没什么两样。所以就让它永远在那吧，反正都一样，那还是自己家待着舒服不是吗。流浪的人或许也并不想流浪啊， 只是没有家可回，只好四海为家咯。

你再看宋笑不就在毅然决然决定去死的“末日”，一转念，开始接受他普通的生活和普通的老婆了吗。做不了全世界的超级英雄，那就做某两个人的超级英雄。年轻时候以为的英雄主义是改变世界，成为了不起的人，后来发现有一种更伟大的英雄主义是接受自己是个普通人，让每天平淡无常的生活过得特别，保护爱的人，让他们的天真得久一点。

《到Y星去》，虽然两个年轻人，大半夜被房东赶出来，挤在青年旅舍的破房间里，但躺在床上还能开个科幻的脑洞，想象一下去外星的生活，回头来发现还是现世生活好，睡一觉明天早上起来继续不知道第几次地找房子搬家。这是一种积极的温暖的和解，好，你厉害，频频给我急风冷雨，但我爬起来继续走啊，并且内心有个地方是一直柔软温暖，任你怎么虐都触碰不到的啊。看谁笑到最后才是赢家，我喜欢这个解法。

还有这篇《西瓜》，我简直看到十年后的我。心高气傲以为自己只要有能力有技术，就可以不看别人的脸色，就可以不受没必要的委屈。一直以来我就特别反感各种成功学，一定要有房有车，嫁个又高又帅有能力的男人，就是人生赢家了吗？哪怕你有理想有抱负，认真做事，真诚善良，也不值一提？张梅不情愿也很尴尬，她挑西瓜超准的老公很无辜，但是为了保住工作她不得不试着妥协，我觉得那就是我自己，看到他们窘迫得提着烟酒西瓜倒几班公交低声下气得给领导打电话还吃闭门羹，我觉得特别委屈，委屈到掉眼泪。但还好，最后文珍还是决定给读者一些安慰，酒洒了，啃完了西瓜，把烟拿出来自己抽，尽管前路不知道如何，但我觉得我们没输，我很骄傲。生活每每把我拉回现实，但我一直尽力捍卫自己的原则和可怜的理想主义，或许偶尔也要做出让步，但只要还在坚持，就没输。

《觑红尘》就有点无奈了，两个骄傲的人那可不就是错过，但我想天雷勾地火的爱情也有一个属性是短暂，它不可能长久，它不属于婚姻不属于生活，是形而上的。一朝拥有过就足够幸运了，想要把它延期保鲜基本没可能。这件事情我还是能心甘情愿的接受的，细水长流更适合我。

后话

看到后记才发现，为什么每一篇都能看到一部分的我，原来我们有很多相似的经历。但我没那么敏感，也更得过且过，所以这个过程比较平缓。我相信每个人都会慢慢的跟生活达成某种君子协议，你退一步我让两尺，双方都不被改变太多，平静安详，好好地。

用hexo和githubio搭建个人博客

嫌免费空间限制太多又觉得自己买域名和空间自己打理太麻烦？可以试一试用github pages和hexo搭建免费个人博客，免费，方便而且还有点酷。我就从自己的实践经历来记录一下从一无所知到成功能用的几个关键步骤。

Hexo

Hexo提供一种的简洁方便的博客框架，使用markdown解析文章并生成静态网页。但要想使用hexo，系统首先要支持Node.js。
Node.js的下载安装。

安装完毕后，就可以在Node.js环境下下载安装Hexo. 具体步骤如下，打开你的terminal

Hexo的几个常用命令及简写为：

  • hexo new “name” / hexo n 用于声称新的post
  • hexo generate / hexo g 生成静态文件
  • hexo server / hexo s 启动本地预览
  • hexo deploy / hexo d 部署到远程（比如下文要讲的githubio）

一般流程是先用hexo n name 在hexo/blog/source/_posts/产生一个name.md。 打开就可以编写内容，对markdown语言不熟悉的同学可参考一些基本markdown语法，编写完毕后保存。hexo g生成文件，hexo s 启动本地server(http://localhost:4000/)，预览无误后，hexo d 部署到远程。

Github Pages

Github Pages是Github提供给用户用来创建网站展示自己github项目的工具。如果你还没有注册安装Github的话，可参考用户手册. 如果你已经是Github常驻用户了，那么只需要创建一个跟你用户名相同的repo: your_user_name.io，注意名字要跟你的用户名相同，否则不适用。

githubio这个repo使用起来跟其他repo没有区别，同样的git clone, commit, push，只是它会根据文件下的index.html，读取相应的css文件后生成网页，在用hexo搭建博客之前，我就用自己写的html和css文件实现了一个网站。那我们用hexo搭建博客的原理也是一样的，hexo d 在deploy的时候也会生成html和css文件，push到github.io，Github根据这些文件来生成网站，当然有些设置需要先完成。

Config and Deploy

在hexo/blog/目录下找到_config.xml, 这个文件里面有些属性要设置一下：
＃Site

  • title 你博客的名字
  • subtitle 副标题，这里可以是个人简介也可以写一下你的座右铭之类的
  • author 你的名字
  • email, keywords, language, timezone可填可不填

＃Writing

  • post_asset_folder这个默认是false，改成true之后，会在你hexo n 的时候同时生成一个与标题同名的文件夹，用于存放post specific assets, 如果你的blog会有很多图片的话最好每篇post单独保存。

观影｜潘金莲窦娥小白菜

##《我不是潘金莲》

觉得电影里面这个点有点意思，就拿来做标题。我对于冯导的这部作品还是很期待的，所以刚一上映又赶上周五，自然颠颠跑去看。看完我打四星，除了左右两对情侣全程叽叽喳喳以外，很不错的观影体验。那我就来闲扯一下。

故事

先说故事，冯导这部几经坎坷的新作改编自刘震云的同名小说，刘震云的作品比如《手机》，《一地鸡毛》，《一句顶一万句》，都很具有故事性，略带点黑色幽默，还比较适合改编成影视作品。电影英文名译作《I’m not Madam Bovary》，所以在外国人的概念里，包法利夫人是跟潘金莲一样的存在？比起潘金莲在中国群众中不可撼动的荡妇毒妇形象，感觉包法利夫人要差一点呢。

故事大纲我就不讲了，累。就是一个法盲农村妇女因为假离婚假戏真做，丈夫跟别人跑了，觉得委屈一级一级告到北京，这其中和之后发生的事情。是挺委屈，我也觉得委屈，而且到影片结尾，这个委屈也还憋着，并没有沉冤得雪。李雪莲告了十几年，各任领导怕了十几年，想尽千方百计，结果却因为被告李玉河的意外去世而画上句号。李雪莲是真的想复婚再离婚吗？在县里市里告状受阻还被关到局子里之后她是打算放弃的，只不过是想从李玉河那里得到解脱，让他承认是假离婚没错，让大家知道自己没有说假话，让自己的委屈不那么委屈。结果又得了个潘金莲的称号。自己第二次放弃，告诉所有人今年不告状的时候，没有人相信她。我自己选择不告状，跟被你们所有人拦着不告状是两回事情，越是阻挠我越要去。第三次被大头软化，打算忘掉仇人，跟喜欢的人过日子的时候，突然又被背叛，你们用这么卑劣的手段不让我告状，那我一定要去！用首长的话来讲，芝麻大的小事，变成了西瓜。而芝麻和西瓜之间，是那么多根本不关心李雪莲是否清白，只想保住或争取自己利益的男人们。你看看电影海报，李雪莲在中间，外面一圈就是这些人。而电影本身也不关心，假离婚和潘金莲都是引子，借李雪莲把这些人和事串起来而已。

人物

主要人物形象都个个典型，也因此有点单一。
马市长：爱思考爱“譬如”成语的萌萌哒的官僚；
郑县长：很怂很爱自我检讨又总是很被动的下级；
王法官：公事公办，虽然害怕乌纱帽不保遇事先撇清自己责任，但也蛮中立的一个角色；
赵大头，李玉河：有点愚昧的普通男人。
李雪莲：一根筋的农村妇女，天不怕地不怕，受了委屈要讨回公道，恨一个人要杀死他，告不赢的话那就不告了吧。
没有人是恶人，但事情还是成了这个样子，为什么呢？请听马市长为您分析。
印象最深的有两段，一是马市长批评教育郑县长和秘书方法不对，感觉也是全场笑点最密集的一段。二是三个人从李雪莲家里出来，马市长中途下车放水，郑县长和秘书一脸懵逼又惊恐，马市长的官腔，郑县长的唯诺，惟妙惟肖。

画面

一开始听说要用圆形方形构图的时候，我是呵呵的，模仿多兰？非要搞点大新闻？但看完才觉得还是有点意义的，比如圆形构图在人物移动的时候就像是个镜头，跟着一起动，比传统的人动景不动多了代入感；再比如，赵大头跟李雪莲在小旅馆不可描述的那一段，你看到画面上发生什么，也知道画面外发生什么，如果是传统的拍法，这里就有点生硬。另外到了北京就变成长方形，据说是因为北京一切都方正讲规则，有些牵强，但也因为画面的变化预示某种不同。影片最后回到标准画面，是因为往事随风，豁然开朗？我不是很同意。

最后

画面都好美啊！美术指导加鸡腿！想知道在哪里拍的然后去实地考察！

Hello World

Hello World!

试了两个晚上终于把githubio和hexo设置好了，换过很多写东西的地方，从最开始的QQ空间（呵呵），到豆瓣，还搞过wordpress，现在决定长期驻扎在此，但愿能坚持下去。

写字的诉求最开始是要源于要倾诉的欲望，后来觉得时常纪录是个值得坚持的好事情，如果还能跟他人分享就更好。就像我以前不喜欢拍照，觉得一切看在眼里记在心里，可是无奈脑容量有限，记忆会慢慢消散，记忆也会骗人。还是俗话说的好

好记性不如烂笔头。

以后会定期纪录学习笔记，读书笔记，电影观后感，游记，或者仅仅是一时心情，任何事情。

一个新的开始！
